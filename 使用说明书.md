## 📱 快速开始（5分钟上手）

### 第一步：打开方式

**方式一：直接用浏览器打开（推荐电脑端体验）**

1. 解压收到的 ZIP 包，得到 `jf_project` 文件夹
2. 双击打开 `index.html`，浏览器会自动跳转到今日计划页面
3. 建议使用 **Chrome / Edge** 浏览器，Firefox 也可以

**方式二：添加到手机主屏幕（推荐手机端体验）**

> ⚠️ 此方式需要先把项目部署到服务器或使用局域网访问。若只是本地使用，直接用浏览器打开即可。

1. 手机浏览器打开项目地址
2. 点击浏览器菜单 →「添加到主屏幕」/「安装应用」
3. 即可像 App 一样使用，支持离线访问

## 📋 页面详细说明

### 1. 今日计划（核心页面）

**打开文件：** `今日计划.html`

这是每天最常用的页面，整合了当天所有需要做的事。

#### 主要功能区域

**① 天气与日期**
- 顶部自动显示当天日期和当地天气（需要联网）
- 天气影响早间运动建议（下雨会给出室内替代方案）

**② 每日节奏时间轴**
- 一天从起床到睡觉的完整安排
- 显示各时间段的任务（晨练、进食窗口、训练、就寝等）

**③ 今日训练计划**
- 显示今天应该做什么训练（拉/推/腿/恢复日）
- 点击动作名称可查看动作演示视频
- 可勾选完成情况

**④ 今日饮食建议**
- 三餐热量分配
- 蛋白质目标提示
- 可记录实际进食情况

**⑤ 日志记录**
- 记录体重
- 记录今天的训练、饮食、睡眠情况
- 写训练笔记

#### 使用方法
1. 每天早上打开，看一眼今日安排
2. 按时间轴执行
3. 晚上睡前填写日志，记录体重和感受

### 2. 减脂全面计划

**打开文件：** `减脂全面计划.html`

全局视角，了解整个减脂方案的逻辑。

#### 主要功能
- **体重目标进度条**：直观显示距目标还有多远
- **体重折线图**：追踪体重变化趋势
- **每日热量目标**：显示你的 TDEE 和缺口设置
- **三分化训练安排**：7 天训练周期总览
- **里程碑奖励**：每减少 5kg 有小奖励提示，保持动力

#### 使用方法
- 每周末看一次，评估本周进度
- 如果连续两周体重无变化，考虑调整饮食或训练量

---

### 3. 训练跟练页面（工作日版）

**打开文件：** `减脂训练日跟练_三分化.html`

针对工作日设计的训练页面，有 B 站视频引导。

#### 三分化训练安排

| 训练类型 | 颜色标识 | 主要肌群 | 安排日 |
|---------|---------|---------|------|
| 拉日 | 🔵 蓝色 | 背、肱二头肌 | 周一、周四 |
| 推日 | 🟡 黄色 | 胸、肱三头肌、肩 | 周三 |
| 腿日 | 🔴 红色 | 腿、臀 | 周五 |
| 恢复日 | 🟣 紫色 | 全身拉伸 | 周日 |
| 有氧日 | 🟢 绿色 | 心肺 | 周二、周六 |

#### 每天的结构
每天分两块训练：
- **早间（7:30-8:00）**：有氧或核心（约 20-30 分钟）
- **晚间（19:00-20:00）**：主训练跟练视频（约 45-60 分钟）

#### 使用视频功能
1. 进入对应天的面板（如「周一」）
2. 找到视频区域，点击「▶ 点击加载视频」
3. 视频来自 B 站，需要联网
4. 按视频节奏跟练即可

#### 切换到休假版
- 页面顶部有「工作日 / 休假日」切换按钮
- 点击即可跳转到休假版页面

### 4. 训练跟练页面（休假版）

**打开文件：** `减脂训练日跟练_休假版.html`

不上班、有大量时间时使用，全天候减脂节奏。

#### 特点
- **更完整的时间安排**：从早上 6:30 到晚上 22:00
- **双倍步数目标**：每天 12,000 步
- **额外散步**：晨间快走 + 晚间散步
- **训练强度相同**：核心力量训练不变

#### 全天时间轴
```
06:30  晨间快走（30分钟，约3000步）
09:00  主训练（与工作日相同内容）
12:00  午餐 + 午休
15:00  下午步行（散步或逛街）
19:00  晚间散步（30分钟）
22:00  睡前拉伸，准备就寝
```

#### 周日特别安排
周日是纯恢复日，不做力量训练：
- 瑜伽拉伸跟练
- 轻度散步
- 睡眠准备（22:00 前入睡）

### 6. 饮食参考

**打开文件：** `饮食参考.html`

备餐前查看，快速规划每餐食材。

#### 功能说明

**① 搜索食材**
- 顶部搜索框直接搜索食材名（如「鸡胸肉」「燕麦」）
- 显示该食材的热量、蛋白质、碳水、脂肪（每 100g）
- 点击「+ 添加」加入当日食材清单

**② 已选食材标签云**
- 已添加的食材显示为标签
- 点击标签上的数字可修改克重
- 点击「×」删除该食材

**③ 周安排结余**
- 右侧显示本周每日的热量/蛋白质结余情况
- 帮你了解哪天吃多了、哪天还有余量

**④ 热量目标**
- 顶部显示今日热量目标和已摄入量
- 一键分配三餐比例

#### 建议用法
1. 周日备餐前打开
2. 搜索你打算买的食材
3. 调整克重，保证蛋白质达标（目标值看今日计划页面）
4. 确保全天热量在目标范围内（±100 kcal）

### 7. 每日必做清单

**打开文件：** `每日必做清单.html`

7 天打卡清单，每天睡前用 2 分钟检查。

#### 清单内容
每天有 6-8 项必做项目：

- [ ] 步数达标（工作日 8000 步 / 休假日 12000 步）
- [ ] 喝够水（每天 2000ml 以上）
- [ ] 蛋白质达标
- [ ] 训练完成（或执行了恢复日方案）
- [ ] 22:45 前上床
- [ ] 记录今日体重

#### 使用方法
- 每天晚上睡前，逐一勾选完成的项目
- 全部完成 = 绿色满分天 ✅
- 每周看一下完成率，了解哪个习惯最难坚持

### 8. 每周小结

**打开文件：** `每周小结.html`

每周日做一次复盘，10 分钟搞定。

#### 包含内容
- 本周体重变化图表
- 本周训练完成情况

- 下周计划调整建议

#### 使用方法
1. 每周日晚上打开
2. 填写本周实际体重数据
3. 回顾各项打卡完成情况
4. 根据提示调整下周计划

## 💾 数据存储说明

### 数据存在哪里？

本 App 的所有数据存储在**浏览器本地（localStorage）**，主要包括：

| 数据 | 说明 |
|------|------|
| `jf_weekly_plan_v3` | 每周训练计划配置 |
| `jf_ingredient_stock` | 常用食材库存 |
| `jf_training_logs` | 训练日志记录 |
| 体重记录 | 每日体重数据 |
| 打卡记录 | 每日清单完成情况 |

### 注意事项

> ⚠️ **重要：** 清除浏览器缓存会导致数据丢失！

- **同一台设备同一浏览器**：数据永久保存，直到手动清除
- **换浏览器/换设备**：数据不会自动同步（除非使用云同步功能）
- **建议**：定期截图记录体重数据，防止意外丢失

### 云端同步（可选）

本 App 支持通过 GitHub Gist 进行云端同步（跨设备数据共享），配置方法：

1. 申请一个 GitHub 账号
2. 生成一个 Personal Access Token（只需 gist 权限）
3. 在今日计划页面，点击顶部「☁️ 同步」图标
4. 输入 Token 即可启用

## ❓ 常见问题

**Q: 打开页面后什么都没有，一片空白？**  
A: 确认使用的是 Chrome 或 Edge 浏览器，不要用 IE。

**Q: 体重数据不见了？**  
A: 可能是清除了浏览器缓存。建议每周截图保存一次体重折线图。

**Q: B 站视频打不开？**  
A: 检查网络连接。或直接去 B 站搜索对应动作名称跟练。

**Q: 页面在手机上看不清楚？**  
A: 本 App 专为手机设计（宽度 500px），在手机浏览器中体验最好。

**Q: 我想修改热量目标/体重目标怎么办？**  
A: 在「减脂全面计划」页面顶部，找到目标设置卡片，点击修改即可。

**Q: 我不是在减脂，可以用这个 App 吗？**  
A: 可以，这套训练框架（三分化+有氧）适合所有健身目标。饮食部分可以根据自己目标调整总热量。

## 📊 营养目标参考

> 以下数值为本 App 当前用户的参考值，可根据自身情况调整。

| 指标 | 目标 | 说明 |
|------|------|------|
| 每日热量 | ~1600 kcal | 训练日可增加 100-150 kcal |
| 蛋白质 | 120-140g/天 | 维持肌肉的关键 |
| 碳水化合物 | 130-160g/天 | 训练日增加 |
| 脂肪 | 40-55g/天 | 不要过低 |
| 饮水量 | 2000ml以上 | 训练日需更多 |

**蛋白质食物参考（每 100g 含量）：**
- 鸡胸肉：31g 蛋白质
- 鸡蛋：13g 蛋白质（约 2 个蛋 = 13g）
- 豆腐：8g 蛋白质
- 希腊酸奶：10g 蛋白质
- 牛肉：26g 蛋白质
