## 1. 项目概览

**产品名称：** 减脂教练（个人健身助手）

**核心定位：** 为任何有体重管理/健身目标的用户定制手机端全流程管理工具，集成每日计划、饮食框架、运动跟练、云端日志四大模块。

**技术栈：** 纯静态 HTML + CSS + Vanilla JavaScript，零框架依赖。

**核心功能：**
- 每日计划（天气感知 + 时间轴 + 训练动作清单 + 晚餐配比）
- 体重追踪（Canvas 画折线图 + 里程碑进度条）
- 每日日志（饮食评价 + 训练/饮水/早睡勾选 + 文字笔记）
- 云端同步（GitHub Gist，跨设备数据共享）
- PWA（可添加到手机主屏幕，离线可用）
- 多页面导航（4 个 Tab 标签页）

**设计哲学：**
- 手机优先（max-width: 500px）
- 深色主题（`#0f0f0f` 背景）
- 三色卡片系统（橙=今日 / 蓝=明日 / 紫=时间轴）
- 所有营养数字使用「食材+克重(营养值 XXg)」格式，杜绝歧义

## 3. 跨人群适配指南

> **本节是本文档区别于普通项目说明书的核心价值所在。**
> 当用户不是"40 岁男性 170cm/85kg 减脂"，而是其他人群时，AI 智能体应根据本节调整所有内容。

### 3.1 体重目标分类

| 场景 | BMI 范围 | 核心策略 | 热量设置 |
|------|---------|---------|---------|
| **大体重减脂** | ≥28（肥胖） | 低冲击运动+严格饮食+关节保护优先 | TDEE - 500~800 |
| **超重减脂** | 24–28 | 可持续运动+饮食控制 | TDEE - 300~500 |
| **正常体重塑形** | 18.5–24 | 力量训练为主+微调饮食 | TDEE 维持或 +100~200 |
| **瘦子增肌** | <18.5 | 热量盈余+渐进超负荷力量训练 | TDEE + 300~500 |
| **瘦胖子（skinny fat）** | 18.5–24 但体脂高 | 身体重组（同时增肌减脂） | TDEE 维持 + 高蛋白 |

### 3.2 性别差异适配

| 维度 | 男性 | 女性 |
|------|------|------|
| **BMR 公式** | Mifflin-St Jeor: `10×W + 6.25×H - 5×A + 5` | `10×W + 6.25×H - 5×A - 161` |
| **必需体脂** | 3–5% | 10–13% |
| **健康体脂范围** | 10–20% | 18–28% |
| **减脂目标体脂** | <15%（腹肌可见） | <22%（马甲线可见） |
| **蛋白质推荐** | 1.6–2.2g/kg瘦体重 | 1.4–2.0g/kg瘦体重 |
| **月经周期考虑** | 无需 | 排卵期力量峰值高8%，黄体期建议碳水稍增 |
| **铁需求** | 8mg/天 | 18mg/天（经期女性） |
| **常见心理模式** | 急于求成→过度训练 | 怕长肌肉→有氧过多+力量不足 |

**⚠️ 特别注意：** 女性不能用总体重算蛋白质！BMI>24 的女性同样应用瘦体重计算。女性瘦体重 ≈ 总体重 × (1 - 体脂率)。若体脂率未知，女性可粗略用 BMI 估算：
- BMI 28+ → 体脂率约 35–40%
- BMI 24–28 → 体脂率约 30–35%
- BMI <24 → 体脂率约 25–30%

### 3.3 年龄适配

| 年龄 | 训练建议 | 营养建议 |
|------|---------|---------|
| **<25** | 恢复快，可用高频训练(5–6次/周) | 蛋白需求可偏高(1.8–2.2g/kg) |
| **25–40** | 标准4–5次/周，注意热身 | 标准配比 |
| **40–55** | 降低到3–4次/周，重视热身+拉伸+关节保护 | 蛋白1.6–1.8g/kg，增加钙和维D |
| **55+** | 2–3次力量+每日散步，防跌倒训练 | 蛋白1.6–2.0g/kg（防肌少症），钙1200mg/天 |

### 3.4 健康问题适配

| 健康问题 | 适配方案 |
|---------|---------|
| **膝关节痛** | 禁止跑步/跳跃/深蹲，改用骑行/游泳/靠墙静蹲/臀桥 |
| **腰椎问题** | 禁止杠铃深蹲/硬拉，改用臀桥/死虫式/鸟狗式/靠墙静蹲 |
| **高血压** | 避免憋气发力(Valsalva)，中等强度有氧为主，监测运动中血压 |
| **糖尿病** | 训练前后监测血糖，碳水摄入需配合训练时间，防低血糖 |
| **肩袖损伤** | 见第 8 节：肩关节损伤与康复 |
| **睡眠呼吸暂停(OSA)** | 见第 7.3 节 |
| **骨质疏松** | 负重训练+高冲击禁用，改用弹力带+自重 |
| **无健康问题** | 按标准方案执行即可 |

### 3.5 可用器械适配

| 器械条件 | 训练方案 |
|---------|---------|
| **零器械（纯自重）** | 全部使用附录B低冲击动作 + 进阶用自重变式 |
| **弹力带** | 自重+弹力带替代所有拉力动作 |
| **哑铃+弹力带** | 标准三分化方案（当前默认） |
| **健身房** | 固定器械优先（易上手），再过渡到自由重量 |

## 4. 人体能量代谢基础

### 4.1 Mifflin-St Jeor 公式（当前最准确的 BMR 估算公式）

```
男性：BMR = 10 × 体重(kg) + 6.25 × 身高(cm) - 5 × 年龄 + 5
女性：BMR = 10 × 体重(kg) + 6.25 × 身高(cm) - 5 × 年龄 - 161
```

该公式 1990 年由 Mifflin 等人发表在《美国临床营养学杂志》，对肥胖人群准确度优于 Harris-Benedict 公式。被美国营养与饮食学会（AND）推荐为首选 BMR 估算方法。

### 4.2 TDEE（每日总能量消耗）

```
TDEE = BMR × 活动系数

活动系数参考：
1.2   — 久坐不动（办公室工作+不运动）
1.375 — 轻度活动（每周运动 1–3 天）
1.55  — 中度活动（每周运动 3–5 天）
1.725 — 高度活动（每周运动 6–7 天）
1.9   — 极高度活动（体力劳动+每天训练）
```

### 4.3 能量缺口/盈余设置

| 目标 | 设置 |
|------|------|
| 减脂 | 每日热量 = TDEE - 300~800 kcal（不应低于 BMR） |
| 维持 | 每日热量 = TDEE |
| 增肌 | 每日热量 = TDEE + 200~500 kcal |

**⚠️ 安全红线：**
- 男性每日摄入不应低于 1500 kcal 长期执行
- 女性每日摄入不应低于 1200 kcal 长期执行
- 国家卫健委指南：男性 1200–1500 kcal，女性 1000–1200 kcal（限能量平衡膳食）

### 4.4 体脂率估算（无专业设备时）

```
从 BMI 粗略估算体脂率：
成年男性：体脂率(%) ≈ (1.20 × BMI) + (0.23 × 年龄) - 16.2
成年女性：体脂率(%) ≈ (1.20 × BMI) + (0.23 × 年龄) - 5.4

瘦体重(kg) = 总体重(kg) × (1 - 体脂率/100)
```

引用来源：Deurenberg 公式（1991），《英国营养学杂志》。

## 5. 营养学核心原理

### 5.1 三大营养素的能量密度

| 营养素 | 能量密度 | 食物热效应(TEF) |
|--------|---------|----------------|
| 蛋白质 | 4 kcal/g | 20–30%（最高，消化蛋白质本身耗能大） |
| 碳水化合物 | 4 kcal/g | 5–10% |
| 脂肪 | 9 kcal/g | 0–3% |
| 酒精 | 7 kcal/g | —（减脂期应限制） |

### 5.2 蛋白质：为什么按瘦体重算

**核心原理：** 体脂肪是代谢惰性组织，不需要蛋白质来维持。只有肌肉、器官、骨骼等瘦体重组织需要蛋白质。

```
正确算法：
每日蛋白质(g) = 瘦体重(kg) × 系数

系数选择：
- 久坐者：0.8–1.0 g/kg 瘦体重（基础需求）
- 减脂期：1.6–2.0 g/kg 瘦体重（防肌肉流失+增加饱腹感）
- 增肌期：1.6–2.2 g/kg 瘦体重
- 大体重减脂新手：1.6–1.8 g/kg 瘦体重（肠胃耐受是瓶颈）

⚠️ 错误做法：用总体重算蛋白质
例：85kg × 2.0g = 170g 蛋白 ≈ 750g 鸡胸肉 → 根本吃不下！
✅ 正确做法：瘦体重 61kg × 1.6 = 98g 蛋白 ≈ 430g 鸡胸肉 → 可行。
```

来源：ACSM 立场声明(2009)，JISSN(2017)，中国营养学会。

### 5.3 蛋白质的饱腹感优势

蛋白质通过以下机制降低总热量摄入：
1. 刺激 CCK（胆囊收缩素）分泌 → 向大脑发送饱腹信号
2. 抑制饥饿素(Ghrelin) → 减少饥饿感
3. 高 TEF(20–30%) → 消化蛋白质本身耗能，净热量更低
4. 稳定血糖波动 → 避免血糖过山车引起的假性饥饿

来源：Bohe 等人（2026），《蛋白质减肥法原理》。

### 5.4 碳水化合物策略

| 目的 | 推荐量 | 说明 |
|------|--------|------|
| 减脂期 | 瘦体重 × 2.0–3.0 g/kg | 训练日偏高，休息日偏低 |
| 维持期 | 瘦体重 × 3.0–4.0 g/kg | 根据活动量调节 |
| 增肌期 | 瘦体重 × 4.0–6.0 g/kg | 保证训练能量+糖原储备 |

**碳水质量优先级：** 全谷物 > 薯类 > 精白米面 > 添加糖

**碳水循环（进阶减脂策略）：**
```
训练日：碳水体重 × 2.5–3.0g（有训练消耗）
休息日：碳水体重 × 1.5–2.0g（无训练消耗）
总周热量缺口不变，但训练日有足够能量保证训练质量。
```

来源：中国居民膳食指南(2022)，ACSM。

### 5.5 常见食物蛋白质含量速查

| 食物 | 100g 蛋白质含量 | 备注 |
|------|---------------|------|
| 鸡胸肉(生) | ~22g | 去皮。高蛋白低脂，性价比之王 |
| 瘦牛肉(生) | ~22g | 含铁+锌+肌酸，增肌佳品 |
| 虾仁(生) | ~18g | 极低脂肪，易消化 |
| 鸡蛋(全蛋) | ~12g/100g = 约 6g/个 | 蛋黄含全部脂溶维生素 |
| 牛奶 | ~3.2g/100ml | 250ml ≈ 8g 蛋白 |
| 豆腐(北豆腐) | ~8g | 植物蛋白，含钙 |
| 三文鱼(生) | ~20g | 含 Omega-3 |
| 猪瘦肉(生) | ~20g | 脂肪含量高于鸡牛 |

### 5.6 常见食物碳水化合物含量速查

| 食物 | 碳水化合物/100g |
|------|----------------|
| 米饭(熟) | ~28g |
| 玉米(熟) | ~22g（半根约 80g ≈ 18g 碳水） |
| 红薯(熟) | ~20g（半个约 100g ≈ 20g 碳水） |
| 南瓜(熟) | ~7g（200g ≈ 14g 碳水，低碳主食首选） |
| 馒头(熟) | ~45g（减脂期应限制） |
| 面条(熟) | ~25g |

### 5.7 进食顺序策略

国家卫健委《成人肥胖食养指南(2024)》推荐：
```
进食顺序：蔬菜 → 蛋白质 → 主食
```
原理：先用蔬菜的纤维素占据胃容量 → 蛋白质增加饱腹感 → 最后吃主食时已经半饱，自然减少碳水摄入量。

## 6. 运动科学基础

### 6.1 ACSM 运动处方核心原则

美国运动医学会(ACSM)是全球运动科学的权威标准，其运动处方遵循 FITT-VP 原则：

| 要素 | 减脂人群 | 增肌人群 |
|------|---------|---------|
| **频率(F)** | 有氧 5–7 天/周 + 抗阻 2–3 天/周 | 抗阻 3–5 天/周 |
| **强度(I)** | 中低强度(HR 60–70% HRmax) | 中高强度(65–85% 1RM) |
| **时间(T)** | 有氧 150–300 分钟/周 | 抗阻 45–60 分钟/次 |
| **类型(T)** | 低冲击有氧 + 全身抗阻 | 分化训练 + 渐进超负荷 |
| **总量(V)** | 周消耗 2000 kcal 以上 | 周组数 10–20 组/肌群 |
| **进阶(P)** | 先增加时长，再增加强度 | 先增加次数，再增加重量 |

来源：ACSM's Guidelines for Exercise Testing and Prescription, 11th Edition.

### 6.2 大体重人群运动安全原则

**绝对禁止：**
- 跑步/跳绳（体重 × 地面反作用力 ≈ 膝关节承受 3–5 倍体重）
- 跳跃类 HIIT（Burpee、跳箱、跳深）
- 高冲击有氧操

**优先推荐：**
1. 骑行（体重由座椅支撑，膝关节零冲击）⭐首推
2. 游泳/水中行走（浮力抵消体重，关节零负荷）
3. 椭圆机（足不离踏板，无冲击）
4. 快走/散步（低冲击，易坚持）
5. 自重力量训练（可控制强度）

**关节保护公式：**
```
关节承受力 = 体重(kg) × 运动冲击系数
冲击系数：骑行=0.3, 步行=1.2, 慢跑=2.5, 跳绳=4.0
```
结论：85kg 大体重者慢跑时膝盖承受 85×2.5≈210kg 冲击，骑行仅 85×0.3≈26kg。

来源：《运动康复医学》大基数无跳跃 HIIT 研究(2025)；Liftyolife — Obesity Exercise: Beginner Workouts(2026)。

### 6.3 三分化训练原理

**创始人/推广者：** B站 UP 主「好人松松」，将全身肌群按**动作功能**分为三组：

```
推日(Push)：胸 + 肩前中束 + 三头肌
    ↓ 所有"推"的肌群在推日训练，利用协同效应"趁热打铁"
拉日(Pull)：背 + 后肩 + 二头肌
    ↓ 所有"拉"的肌群在拉日训练
腿日(Legs)：臀腿 + 核心
    ↓ 下肢大肌群单独一天，对心肺要求最高
```

**为什么要三分化而不是五分化（胸/背/肩/臂/腿各一天）？**
1. 新手每次练一个部位效率低，三分化每次练多个肌群，一周每个肌群可刺激 2 次
2. 每个肌群每周 2 次刺激 > 1 次刺激（肌肉蛋白合成高峰在训练后 24–36h）
3. 单次训练 45–60 分钟，21 组左右，适合业余健身者
4. 训练频率灵活：不是固定周一必须拉日，而是"推→拉→腿→休息→推→拉→腿"无限循环

**组间休息规范：**
- 大肌群（胸/背/腿）：休息 2–3 分钟
- 小肌群（肩/臂）：休息 1.5–2 分钟
- 腿日可适当延长（对心肺要求高）

来源：好人松松 B站教程；知乎专栏「健身减肥的理论学习之三分化训练计划」(2023)。

### 6.4 训练进阶原则

```
新手阶段（前 3 个月）：神经适应为主
  → 学习动作模式 → 固定器械 → 自由重量
  → 重量不重要，动作质量第一

中级阶段（3–12 个月）：渐进超负荷
  → 先增加次数(8→10→12) → 再增加组数 → 最后增加重量
  → 每次增加不超过 5% 重量

高级阶段（12 个月+）：周期化训练
  → 力量周期 × 3 周 → 肌肥大周期 × 4 周 → 减载周 × 1 周
```

### 6.5 运动频率指南

```
每周 1–2 次：聊胜于无
每周 3–4 次：有效但不充分
每周 4–5 次：最佳（足够的刺激+充足的恢复）
每周 6 次：频率极限
每周 7 次：自残（无恢复日，受伤风险极高）
```

每周至少需要 1 天完全休息，让中枢神经系统和肌肉修复。

来源：好人松松三分化教程。

## 7. 睡眠与激素调控

### 7.1 三大减脂激素

| 激素 | 功能 | 如何优化 |
|------|------|---------|
| **皮质醇**(Cortisol) | 应激激素。过高→分解肌肉+储存腹部脂肪+增加食欲 | 充足睡眠、不要过度训练、晚餐适量碳水稳定夜间血糖 |
| **生长激素**(GH) | 深度睡眠时脉冲性分泌→分解脂肪+修复肌肉 | 22:00–02:00 间入睡（抓住最早几个 GH 脉冲） |
| **饥饿素/瘦素**(Ghrelin/Leptin) | 饥饿素让你饿，瘦素让你饱 | 睡眠<5h → 饥饿素↑28% + 瘦素↓18% |

来源：Spiegel 等人(2004)，《内科学年鉴》。

### 7.2 黄金睡眠守则

```
1. 22:00 电子宵禁：蓝光抑制褪黑素分泌长达 90 分钟
2. 完全黑暗：遮光窗帘+眼罩（任何光线都会抑制褪黑素）
3. 固定作息：每天同一时间睡/起（含周末），稳定昼夜节律
4. 睡前不饱食：最后一餐距睡觉 ≥ 2.5 小时
5. 睡眠时长：5 个完整周期 × 90 分钟 = 7.5 小时
6. 睡前停水：睡前 2 小时停止饮水，避免夜尿打断深睡
```

### 7.3 OSA（阻塞性睡眠呼吸暂停）与减脂

**流行病学数据：**
- 全球约 9.36 亿成年人患有 OSA（30–69 岁）
- 中国 OSA 患者约 1.76 亿
- BMI > 28 者 OSA 患病率是正常体重者的 5–7 倍

**OSA 如何阻碍减脂（恶性循环）：**
```
BMI 高 → 气道脂肪堆积 → 夜间反复缺氧
    → 皮质醇持续升高
    → 饥饿素↑ + 瘦素↓
    → 白天更想吃（尤其是高糖高油）
    → 更难减脂
    → BMI 继续高（恶性循环）
```

**打破循环的关键：**
- 减重本身是最有效的 OSA 治疗：减 10% 体重可改善 20–40% 的 AHI
- 侧卧睡姿：使 AHI 降低 50%（背后放枕头防止翻身）
- CPAP（持续气道正压通气）：标准治疗，但依从性仅 40%
- 禁酒（酒精松弛咽部肌肉，加重塌陷）

来源：JLGH 期刊 OSA 与肥胖综述；中国 OSA 诊疗指南；ConsultantLive 跨学科减重干预 OSA 研究(2026)。

## 8. 肩关节损伤与康复

### 8.1 肩袖撞击综合征 (Shoulder Impingement Syndrome)

**病因：** 肩峰下空间变窄 → 肩袖肌腱（主要是冈上肌）被挤压 → 炎症 → 疼痛 → 更肿 → 更挤（恶性循环）

**高危人群：** 游泳、棒球、排球、网球运动员；建筑/油漆/装修工人；40 岁以上人群。

**典型症状（来自 Cleveland Clinic, 2025）：**
- 举手过头时前肩疼痛（90% 案例）
- 手伸到背后（如解内衣扣）引发锐痛（特异度最高）
- 夜间痛，侧卧患侧加重
- 手臂抬起到 60°–120° 弧段（"疼痛弧"）疼痛

### 8.2 绝对禁止的动作

```
🚫 禁止动作（会加剧肩峰撞击）：
1. 肩上推举（杠铃/哑铃推举）—— 最核心禁忌
2. 宽距俯卧撑（肩外展 > 45°）
3. 哑铃飞鸟（大角度外展+外旋）
4. 颈后下拉/颈后推举（肩关节极度外旋）
5. 任何过顶动作（overhead press、倒立撑等）

替代方案：窄距/墙壁俯卧撑（肘贴身）、前平举（掌心相对，抬至肩高即停）
```

### 8.3 康复核心动作（来自 Cleveland Clinic 物理治疗方案）

```
每日必做（早晚各一次，3 分钟）：
1. 弹力带肩外旋 2×15
   → 弹力带固定于门把手，患侧肘贴身体，前臂向外旋转
   → 这是肩袖撞击康复的"王牌动作"
2. 肩绕环 10 圈/方向（小幅度，无痛范围）
3. 墙壁天使 10 次（背靠墙，手臂贴墙上下滑动）
```

### 8.4 恢复时间线

```
急性期（第 1–2 周）：休息 + 冰敷 + NSAIDs，完全停止诱发动作
恢复期（第 2–6 周）：物理治疗，逐步增加活动范围
功能性恢复（第 6–12 周）：恢复力量训练，但永久避开禁止动作
完全恢复：可能需要 3–12 个月

⚠️ 在医生/物理治疗师批准前，不要恢复运动！
过早恢复→再损伤风险 → 可能发展成肩袖撕裂 → 需要手术。
```

来源：Cleveland Clinic — Shoulder Impingement Syndrome (2025)；AAOS 肩袖撞击指南(2024)。

## 9. 中国居民膳食指南摘要

### 9.1 国家卫健委《体重管理指导原则(2024)》

**核心数据：**
- 我国成年居民超重率 34.3%，肥胖率 16.4%
- 2030 年预测超重肥胖率将达 70.5%
- 科学减重目标：6 个月内减少当前体重的 5%–10%
- 合理减重速度：每月 2–4 kg

**三大营养素供能比（国家推荐）：**
```
碳水化合物：50%–65%（总热量来源）
脂肪：20%–30%
蛋白质：10%–20%
```

### 9.2 国家卫健委推荐饮食守则

```
✅ 优先选择：
- 全谷物（替代精白米面）
- 足量新鲜蔬菜（减少高糖水果和高淀粉蔬菜）
- 低脂蛋白质（去皮鸡胸、鱼虾、瘦肉）
- 低脂/脱脂奶类

❌ 限制：
- 油炸食品、含糖糕点、糖果、肥肉
- 食盐 < 5g/天
- 烹调油 < 20–25g/天
- 添加糖 < 25g/天
- 严格限酒（每克酒精产生 7kcal）

⏰ 进餐建议：
- 早中晚供能比 3:4:3
- 晚餐 17:00–19:00
- 细嚼慢咽
- 进食顺序：蔬菜→肉类→主食
```

### 9.3 中国居民膳食宝塔核心推荐

| 食物类别 | 每日推荐量 |
|---------|-----------|
| 谷薯类 | 250–400g（全谷物+杂豆 50–150g，薯类 50–100g） |
| 蔬菜 | 300–500g（深色蔬菜占一半） |
| 水果 | 200–350g |
| 动物性食物 | 120–200g（每周至少 2 次水产品） |
| 奶及奶制品 | 300–500ml |
| 大豆及坚果 | 25–35g |
| 油盐 | 油 25–30g，盐 < 5g |

来源：中国营养学会《中国居民膳食指南(2022)》；国家卫健委《成人肥胖食养指南(2024)》。

### 9.4 睡眠建议（来自国家卫健委指南）

- 保证每日 7 小时左右的睡眠时间
- 按时作息，遵循昼夜生物节律
- 睡眠不足可导致"过劳肥"（内分泌紊乱+脂肪代谢异常）

## 10. 减重医学临床共识

### 10.1 肥胖诊断标准（中国标准）

```
BMI < 18.5：体重过低
BMI 18.5–23.9：正常
BMI 24.0–27.9：超重
BMI ≥ 28.0：肥胖

腰围标准：
男性腰围 ≥ 90cm → 中心性肥胖
女性腰围 ≥ 85cm → 中心性肥胖
```

### 10.2 国际肥胖运动处方共识

| 指南来源 | 核心建议 |
|---------|---------|
| **ACSM** | 有氧 150–300 分钟/周 + 抗阻 2–3 次/周，周消耗 ≥ 2000 kcal |
| **WHO** | 成人每周 150–300 分钟中强度或 75–150 分钟高强度有氧 + 2 次力量训练 |
| **中国卫健委** | 中低强度有氧为主 + 抗阻为辅，150–300 分钟/周，每周运动消耗 2000 kcal 以上 |
| **Cochrane 综述** | 饮食+运动联合干预比单一干预减重效果多 20% |

### 10.3 减重后反弹预防

```
反弹的核心机制：
1. 代谢适应：减重后 BMR 下降 15–25%（身体进入"节能模式"）
2. 激素变化：饥饿素永久性↑，瘦素永久性↓（减重后饥饿感更强）
3. 肌肉流失：未补足蛋白质 → 减掉的不只是脂肪还有肌肉 → BMR 更低

预防策略：
1. 高蛋白饮食（维持肌肉量 = 维持 BMR）
2. 力量训练（直接对抗肌肉流失）
3. 渐进减重（不追求速成 → 1–2kg/月是肌肉保护速度）
4. 过渡期（到达目标后继续追踪 3–6 个月再放松）
```

### 10.4 蛋白质与减重后肌肉保护

```
高蛋白组(1.6g/kg，减重10kg)：肌肉仅流失 0.5kg
低蛋白组(0.8g/kg，减重10kg)：肌肉流失 3kg
结果：高蛋白组半年后反弹率降低 4 倍

数据来源：中国居民蛋白质摄入研究（2025）。
```

## 26. 公式速查库

> 本节汇总本文档中所有公式，方便 AI 智能体一次性查阅。

### BMR 计算

```
Mifflin-St Jeor 公式（推荐首选）：

男性：BMR = 10 × W + 6.25 × H - 5 × A + 5
女性：BMR = 10 × W + 6.25 × H - 5 × A - 161

其中：W=体重(kg), H=身高(cm), A=年龄
```

### TDEE 计算

```
TDEE = BMR × 活动系数
活动系数：1.2(久坐) / 1.375(轻度) / 1.55(中度) / 1.725(高度) / 1.9(极高度)
```

### BMI 计算

```
BMI = 体重(kg) / (身高(m))²
中国标准：<18.5 偏瘦 / 18.5–23.9 正常 / 24–27.9 超重 / ≥28 肥胖
```

### 体脂率估算（无专业设备）

```
Deurenberg 公式(1991)：
男性：体脂率(%) ≈ 1.20 × BMI + 0.23 × 年龄 - 16.2
女性：体脂率(%) ≈ 1.20 × BMI + 0.23 × 年龄 - 5.4

瘦体重(kg) = 总体重(kg) × (1 - 体脂率/100)
```

### 营养素计算

```
蛋白质(g/天) = 瘦体重(kg) × 系数
  减脂：1.6–2.0 | 增肌：1.6–2.2 | 维持：1.2–1.6

碳水(g/天) = 瘦体重(kg) × 系数
  减脂：2.0–3.0 | 增肌：4.0–6.0

脂肪(g/天) = 瘦体重(kg) × 系数
  一般：0.6–1.0
```

### 热量计算

```
蛋白质热量 = 蛋白克数 × 4
碳水热量 = 碳水克数 × 4
脂肪热量 = 脂肪克数 × 9
总热量 = 蛋白热量 + 碳水热量 + 脂肪热量
```

### 减重速度

```
安全减重速度：每月 2–4 kg（每周 0.5–1 kg）
初期目标：6 个月内减少当前体重 5%–10%
热量缺口：每日 300–800 kcal，但总摄入不低于 BMR
```

### 关节冲击估算

```
关节承受力 ≈ 体重(kg) × 运动冲击系数
骑行：0.3 | 步行：1.2 | 慢跑：2.5 | 跳绳：4.0
```

## 附录 E：三分化训练原理详解

> 本节为好人松松三分化体系的核心知识点摘要。

### E.1 体系核心逻辑

将全身肌肉按**动作功能**分为三组，每次训练利用肌群的协同效应"趁热打铁"：
- **推日(Push)：** 胸 + 肩前中束 + 三头肌 → 所有"推"的肌群
- **拉日(Pull)：** 背 + 后肩 + 二头肌 → 所有"拉"的肌群
- **腿日(Legs)：** 臀腿 + 核心 → 下肢大肌群

### E.2 组数分配

- 大肌群 9 组（可选择 3 个动作，各 3 组；或 2 个动作各 4–5 组）
- 两个小肌群各 6 组
- 合计约 21 组/次，训练时长约 60 分钟
- 腿日因只练一个部位，组数稍多

### E.3 组间休息

- 大肌群（胸/背/腿）：2–3 分钟
- 小肌群（肩/臂）：1.5–2 分钟
- 腿日对心肺要求高，可适当延长

### E.4 训练频率

- 不固定周几练什么，而是"推→拉→腿→休息"循环
- 每周每个肌群可刺激 2 次（优于五分化方案）
- 每周 4–5 次最佳，6 次为极限，7 次为自残

### E.5 动作选择原则

- 动作库按模式分类，训练者在同类动作中自行选择
- 新手先易后难：固定器械 → 自由重量
- 练背日可 2 下拉+1 划船或 2 划船+1 下拉
- 胸以推为主，夹为辅；肩前束以推举为主，前平举为辅
